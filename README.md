# React JS Assignment: Simple Student Management System

## Objective
Build a basic React application to manage a list of students and understand state management, props, and component interaction.

---

## Concepts Covered
- Functional Components
- `useState` Hook
- Props
- Forms & Input Handling
- Lists & Keys
- Conditional Rendering

---

## Project Features
- Add a student (Name, Age, Course)
- Display a list of students
- Delete a student
- Search students by name (real-time)
- Display total number of students

---

## Suggested Components
| Component Name | Description |
|----------------|------------|
| App | Main component, manages global state |
| StudentForm | Form to add new students |
| StudentList | Displays list of students |
| StudentCard | Displays individual student details |
| SearchBar | Filters students by name |

---

## Functional Requirements
- Students are stored in React state
- Each student has: `id`, `name`, `age`, `course`
- Form validation to prevent empty fields
- Real-time search filtering

---

## UI Requirements
- Simple card layout for each student
- Clean form design
- Basic responsive layout
